### Vulnerabilities

This page contains a summary of Common Vulnerabilities and Exposures (CVE) discovered and fixed 
in Apache Mynewt.

| <a href="https://www.cve.org/CVERecord?id=CVE-2025-62235">CVE-2025-62235 </a> | Incorrect handling of SMP Security Request could lead to undesirable pairing | NimBLE through 1.8.0 | NimBLE 1.9.0  |
