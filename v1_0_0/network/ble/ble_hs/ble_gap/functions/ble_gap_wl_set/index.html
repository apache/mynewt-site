ble_gap_wl_set

int
ble_gap_wl_set(
    const struct ble_gap_white_entry *white_list,
                             uint8_t  white_list_count
)

Description

Overwrites the controller's white list with the specified contents.

Parameters

Parameter Description
white_list The entries to write to the white list.
white_list_count The number of entries in the white list.

Returned values

Value Condition
0 Success.
Core return code Unexpected error.
